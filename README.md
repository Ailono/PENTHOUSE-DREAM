# 🏠 Penthouse Dream

Экономическая RPG-игра: Telegram-бот + Desktop-приложение.

## 🚀 Запуск Telegram бота (пошагово)

### 1. Установи Python 3.10+

### 2. Открой терминал в папке `backend`

```bash
cd penthouse-dream/backend
```

### 3. Установи зависимости

```bash
pip install -r requirements.txt
```

### 4. Получи токен бота

Напиши [@BotFather](https://t.me/BotFather) в Telegram, создай нового бота командой `/newbot`, скопируй токен.

### 5. Создай файл `.env`

Скопируй `.env.example` в `.env` и вставь токен:

```
BOT_TOKEN=1234567890:ABCdefGHIjklmNOPqrstUVwxyz
```

Остальное менять не нужно (по умолчанию SQLite, база создаётся автоматически).

### 6. Запусти бота

```bash
python -m bot.main
```

Готово! Напиши `/start` своему боту в Telegram.

### Опционально: API + Desktop

```bash
# Отдельный терминал — API сервер
uvicorn app:app --reload --port 8000

# Отдельный терминал — Desktop
cd desktop
npm install
npm start
```

## Команды бота

| Кнопка | Что делает |
|--------|-----------|
| `/start` | Регистрация, приветствие |
| `💰 Баланс` | Показать монеты |
| `🏪 Магазин` | Купить предметы (4 редкости) |
| `💼 Работа` | 9 мини-игр |
| `🎰 Казино` | Рулетка (11 типов ставок) |
| `📸 Фотоохота` | Задания на фото |
| `🎒 Инвентарь` | Посмотреть/экипировать предметы |
| `📊 Статистика` | Прогресс, ачивки, улучшить комнату |

## Что внутри

- **11 типов ставок** в рулетке (европейское колесо, 37 чисел)
- **9 мини-игр**: кликер, уборка, тамагочи, лотерея, крафт, курьер, майнинг, ночной клуб, фотоохота
- **13 достижений** с наградами
- **15 видов мебели** с 4 уровнями редкости
- **Desktop**: 2D комната с анимациями (рыбки, огонь, фонтан, свечи)
- **SQLite** (по умолчанию) или PostgreSQL
